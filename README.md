# Hybrid-IAM-Lab
This project demonstrates a hybrid Identity and Access Management (IAM) environment by integrating an on-premises Active Directory domain with a cloud identity provider. The lab simulates real-world enterprise identity workflows including user provisioning, role-based access control (RBAC), single sign-on (SSO), and multi-factor authentication MFA
